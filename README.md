## Getting Started

To get a local copy up and running follow these simple example steps within your CLI.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/daniel-jacks/todo-app.git
   ```
3. Initialize git repo
   ```sh
   npm init -y
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
